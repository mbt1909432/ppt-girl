Skills Documentation Folder
============================

This folder contains project experience documentation and knowledge base files.

**Purpose:**
- Store project experience summaries and technical documentation
- Document workflows, procedures, and best practices (e.g., `ADDING_NEW_CHARACTER.md`)
- Serve as context reference for future modifications and development

**Usage:**
- When modifying related features, reference these documents as context
- Keep documentation up-to-date as the project evolves
- Use clear, descriptive filenames for easy reference

**Example:**
- `ADDING_NEW_CHARACTER.md` - Step-by-step guide for adding new characters to the frontend

These documents help maintain consistency and provide clear guidance for development tasks.
